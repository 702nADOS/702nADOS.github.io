# Install

## Prerequisites

- Ubuntu 16.04

- git

- [oracle virtualbox](https://www.virtualbox.org)

- [vagrant](https://www.vagrantup.com)

## Quick Steps

```sh
$> git clone https://github.com/argos-research/operating-system.git
$> cd operating-system
$> git submodule init
$> git submodule update
$> vagrant up
```
